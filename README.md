# Miya Protfolio

Personal portfolio (bilingual 中文/EN, single static page — no build step).

## Deploy
- **GitHub Pages:** push these files to the repo root on `main`, then Settings → Pages → Source: *Deploy from a branch* → `main` / `/ (root)`.
- **Netlify/Vercel:** drag this folder onto the deploy dropzone — no config needed.

Entry point: `index.html`. Images live in `assets/`.
